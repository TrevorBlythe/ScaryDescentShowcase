Scary Descent VR

Solo/Multiplayer horror VR!

Scary Descent doesn't fake jumpscares. All deaths in the game happen in real time, no screen pop-ups, no teleport traps. The horror unfolds naturally in the environment, with threats that physically move, stalk, and interact within the same space as you You will feel fear, playing this game. This game is not recommended for people with epilepsy / heart problems.

I'm actively trying to fix all the bugs, send suggestions and bug reports via GitHub Issues!
